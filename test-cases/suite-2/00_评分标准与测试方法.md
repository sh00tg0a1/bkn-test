# 供应链 BKN 测试集 · 评分标准与测试方法

> 版本: v1.0（Suite-2）

---

## 一、评分原则

本套测试集的评分标准非常简单：**以答案是否正确为准**。

- **正确**：被测回答中**所有数值**与**所有要素**均与金标标准答案一致。
- **数值**：包括数量、金额、日期、比例、百分比、单号、物料编码等一切出现在标准答案中的数字与日期类信息。
- **要素**：包括结论、状态、对象名称、关系、是否满足条件等标准答案中明确给出的要点。

---

## 二、关于「可选字段/数据项」的说明

标准答案中有时会附带**便于核对或说明**的字段标识或编码（例如仓库编码 `WH001`、字段名 `has_cold_storage=否` 等）。**若题目本身未要求必须列出具体数据项**，则：

- 被测回答**不必**写出这些编码或字段名；
- 只要**语义与事实**与标准答案一致（例如说明「从深圳天翼中央成品仓库发出」「该仓库无冷库」），即视为**正确**。

**示例**

- 金标：`从深圳天翼中央成品仓库（WH001）发出；该仓库无冷库（has_cold_storage=否）`
- 若被测回答未包含 `WH001`、`has_cold_storage`，但表达了「从深圳天翼中央成品仓库发出」「无冷库」等一致含义，**仍判为正确**。

---

## 三、不合格情形

- 数值与标准答案不一致（含四舍五入方式导致与金标冲突时，以金标为准）。
- 遗漏标准答案中的关键要素，或结论与事实相反。
- 虚构标准答案中未出现的单号、物料、客户等事实。

---

## 四、测试方法（建议流程）

1. **加载知识网络**：按项目要求使用 KWeaver 等工具与知识网络交互。
2. **按题型文件顺序执行**：`questions/` 下对应 `S1`～`S4` 与 `cases`/`answers` 中题号一致。
3. **记录被测回答**：保存到 `result/` 或测试报告中。
4. **逐题对照** `answers/` 中的**涉及对象、关键字段、标准答案**，按本文件第一节、第二节判定正误。

---

## 五、测试报告规范

### 5.1 输出位置

所有测试报告统一输出到项目根目录下：

```
test_report/suite-2/
```

### 5.2 报告文件结构

```
test_report/suite-2/
  总体报告.md                     # 汇总报告（必须）
  S1_直接对象问答_测试报告.md      # 分组报告
  S2_2跳问答_测试报告.md
  S3_3跳问答_测试报告.md
  S4_4跳问答_测试报告.md
```

### 5.3 分组报告（每组 Case 报告）

每个分组报告 **必须** 包含：

| 内容 | 说明 |
|------|------|
| 题号 | Q01–Q25 |
| 被测回答 | Agent 实际返回的回答内容 |
| 标准答案 | 来自 `answers/` 的金标答案 |
| 判定结果 | ✅ 正确 / ❌ 错误 |
| 错误说明 | 若判定为错误，说明与金标答案的具体差异 |
| 工具调用次数 | 该题 Agent 的工具调用次数（即 SQL 查询次数） |
| Token 消耗 | 该题的 Token 使用量（prompt + completion） |

每组报告末尾汇总该组统计：

| 统计项 | 说明 |
|--------|------|
| 正确题数 / 总题数 | 例如 20/25 |
| 该组准确率 | 例如 80% |
| 该组总工具调用次数 | 所有题目工具调用次数之和 |
| 该组总 Token 消耗 | 所有题目 Token 之和 |

### 5.4 总体报告

`总体报告.md` **必须** 包含以下内容：

#### A. 总体统计

| 统计项 | 说明 |
|--------|------|
| 总题量 | 100 题 |
| 正确题数 | 判定为 ✅ 的题目数 |
| 总体准确率 | 正确题数 / 总题数 |
| 总工具调用次数 | 全部 100 题的工具调用次数之和 |
| 平均每题工具调用次数 | 总调用次数 / 100 |
| 总 Token 消耗 | 全部 100 题的 Token 使用量之和 |
| 平均每题 Token 消耗 | 总 Token / 100 |

#### B. 分组得分

| 题型 | 正确 / 总数 | 准确率 | 工具调用次数 | Token 消耗 |
|------|-------------|--------|-------------|-----------|
| S1 直接对象问答 | x/25 | xx% | n | t |
| S2 2跳问答 | x/25 | xx% | n | t |
| S3 3跳问答 | x/25 | xx% | n | t |
| S4 4跳问答 | x/25 | xx% | n | t |
| **合计** | **x/100** | **xx%** | **N** | **T** |

#### C. 问题分析

- 列出所有错误题目及其错误原因分类（如数值偏差、要素遗漏、事实虚构等）。
- 按题型分析错误分布，指出薄弱环节。
- 给出改进建议。
