**Q23**
**提问者**: 领导
**问题**: 今天有什么新的风险吗？
**场景**: S4 风险预警入口

**参考答案要点**:
- 最高风险：T01-000080 的PCB（134-000396）已进入**红色预警**——今天是最晚下采购申请的截止日
- 计算依据：采购交期35天，今天3月21日 + 35天 = 4月25日到货，距交货4月30日仅剩5天装配时间
- 晚一天后果：每晚1天，装配缓冲就少1天；若超过今天下单，可能无法按期完成3000套
- 次要风险：948-000077 全部601条MRP未投放，每天不推进都是在积累风险
- 引导：需要我帮你准备一份给供应商的紧急协调函吗？

**不合格回答特征**: 说"暂时没有新风险"；不能量化"今天是最晚截止"的依据；没有紧迫感

**问题**: T01-000080 有没有到货超期的风险？
**场景**: S4 → 到货超期预警

**参考答案要点**:
- 有超期风险的物料：PCB（134-000396）——如果今天才下PR，4月25日到货，距交货仅5天，任何生产延误都会导致最终入库超期

**不合格回答特征**: 不区分不同物料的超期风险程度；把"PO正常"等同于"不会超期"

**Q25**
**提问者**: PMC
**问题**: 948-000077 的MRP什么时候运算的？为什么一个PR都没有？
**场景**: S4 → MRP执行状态解释

**参考答案要点**:
- MRP运算时间：**2026年3月18日**（3天前），批次号 MRP-20260318-00000114
- 零PR原因：MRP运算后需要PMC手动在ERP中将计划订单"投放"，才会生成采购申请（PR）；当前601条全部停留在"未投放"状态，PMC还没有执行这一步
- 这是正常流程中的一个必要操作环节，不是系统故障
- 需要的行动：PMC登录ERP，找到该批次MRP（MRP-20260318-00000114），按物料交期长短批量投放外购件
- 建议优先处理：交期30天以上的外购件需在本周内完成投放和PR下达
- 引导：需要我帮你筛选出交期最长的物料优先处理吗？

**不合格回答特征**: 说"系统没有数据"；解释不清楚MRP投放和PR的关系；不给出下一步操作引导

**Q26**
**提问者**: 领导
**问题**: 600套T01-000080的工单为什么还没有领料？
**场景**: S4 → 工单执行异常分析

**参考答案要点**:
- 工单情况：KSHA-20260312-006，600套成品，状态"开工/未领料"，计划开工4月29日
- 可能原因：计划开工日是4月29日，当前3月21日，尚未到开工时间，工单显示"开工"但领料还未触发是正常的
- 另一种可能：物料尚未备齐，仓库无法发料
- 建议核实：确认这张工单的物料是否已在仓库备好；关注PCB等尚未采购的物料，这是影响该批次领料的直接卡点
- 引导：需要查看这批工单对应的物料清单吗？

**不合格回答特征**: 直接判断"工单有问题"；不考虑计划开工日还未到的情况；不把领料问题和物料采购状态关联起来

**Q27**
**提问者**: 产品经理
**问题**: 我们现在在追的产品里，有没有哪个已经确定会延期？
**场景**: S4 → 延期风险判断

**参考答案要点**:
- T01-000080（4月30日）：**存在延期风险，但尚未确定**。取决于今天PCB是否下单，以及供应商能否如期交货；若今天下单且供应商按时交货，理论上可以完成，但几乎没有缓冲
- 948-000077（5月20日）：**延期风险正在积累**。MRP全未启动，若本周内不推进采购，60天后必然出现物料缺口；当前还不能说"确定延期"，但如果本周内不行动就会越来越难
- 结论：两个产品都处于"风险中"而非"确定延期"；主动行动可以避免延期
- 行动建议：今天处理T01-000080的PCB，本周内启动948-000077的采购
- 引导：需要我帮你起草一份给相关责任人的提醒通知吗？

**不合格回答特征**: 只说"不确定"；或过于悲观直接说"会延期"；不提出具体改变结果的行动

**Q28**
**提问者**: 领导
**问题**: 948-000077 有没有库存可以先用？
**场景**: S4 → 成品库存与预测需求对比

**参考答案要点**:
- 有库存：62套，全部在**借用仓**（非生产可用库存）
- 借用仓性质说明：借用仓中的物品通常是借调给外部或项目的临时存放，不能直接用于新订单出货
- 预测需求300套，即使借用仓可用，也仅满足62/300=20.7%，缺口238套
- 结论：不能依赖库存，仍需完整推进300套的生产计划
- 建议：确认62套借用仓库存的状态和归属，评估是否可提前退还纳入本次预测交货
- 引导：要进一步了解948-000077的物料备料状态吗？

**不合格回答特征**: 直接说"有62套可以用"；不解释借用仓的含义；忽略与300套需求的差距
