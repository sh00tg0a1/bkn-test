## 4. 主要问题归纳（根因）

### 4.1 数据模型与 CSV「关键数据来源」不一致

| 现象 | 影响题号 | 说明 |
|------|----------|------|
| 无 **在库/在途/冻结** 分项库存 | F1-01 | CSV 写 WMS 台账；库表更接近「库存快照 + 发运」，**无法严格还原 WMS 三态**。 |
| 无 **安全库存** 字段 | F1-04 | 无法答「是否低于安全库存水位」。 |
| 预测单 **无草稿/已确认** 状态 | F1-02 | 仅有 `status=Active` 等，与 ERP 生命周期不一致。 |
| **PR→PO** 外链缺失 | F2-04 | `purchase_order_event.srcbillid` 常为空，**无法自动对账「未转 PO」**。 |
| 无 **工艺路线 / 整机提前期** 统一表 | F3-01、F3-04 | **最早可交付日、周期拆解** 缺参。 |
| **替代料、供应商现货** 未在本次 SQL 中系统展开 | F3-03、F4-02/03 | CSV 要求的能力依赖额外主数据与算法。 |

### 4.2 测试范围与方法（非数据问题）

| 现象 | 影响题号 | 说明 |
|------|----------|------|
| F4 **已补** [`f4_deliverables.md`](./f4_deliverables.md) | F4 全部 | 飞书/网页/简报体 + 来源标注；与 BKN 对比时口径对齐。 |
| 部分题 **未跑完 SQL**（分级、齐套率） | F2-06、F3-03 | 属 **测试执行深度**。 |

### 4.3 口径与业务表述风险

| 风险 | 题号 | 说明 |
|------|------|------|
| `assembly_set` 代理 **自制/采购件个数** | F1-05 | 可能与 ERP **物料类型** 定义不一致。 |
| 发运「运输中」**≠** **在途库存** | F1-01 | 易误导，报告已提示，对照 CSV 仍算 **口径风险**。 |
| 缺料 **「颗」** = MRP **行** 是否等价 | F2-02 等 | 若同一物料多行拆分，需确认是否合并为「颗」。 |
