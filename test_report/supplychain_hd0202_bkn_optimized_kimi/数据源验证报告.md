## 附录 A：查询脚本

### A.1 预测单查询

```sql
SELECT billno, material_number, qty, startdate, enddate, creator_name, auditor_name
FROM public_gitea_analytics.erp_mds_forecast
WHERE billno IN ('YCD2026022800000048', 'YCD2026031600000056');
```

### A.2 MRP 查询

```sql
SELECT 
  rootdemandbillno,
  COUNT(*) as mrp_count,
  COUNT(CASE WHEN materialattr_title = '自制' THEN 1 END) as self_made,
  COUNT(CASE WHEN materialattr_title = '外购' THEN 1 END) as purchase,
  COUNT(CASE WHEN materialattr_title = '委外' THEN 1 END) as outsource
FROM public_gitea_analytics.erp_mrp_plan_order
WHERE rootdemandbillno IN ('YCD2026022800000048', 'YCD2026031600000056')
GROUP BY rootdemandbillno;
```

### A.3 PR 查询

```sql
SELECT pr.billno, pr.material_number, pr.material_name, pr.qty, pr.srcbillnumber
FROM public_gitea_analytics.erp_purchase_request pr
JOIN public_gitea_analytics.erp_mrp_plan_order mrp ON pr.srcbillnumber = mrp.billno
WHERE mrp.rootdemandbillno = 'YCD2026022800000048'
AND mrp.materialattr_title = '外购';
```

### A.4 PO 查询

```sql
SELECT DISTINCT po.billno, po.material_number, po.material_name, po.qty
FROM public_gitea_analytics.erp_purchase_order po
JOIN public_gitea_analytics.erp_purchase_request pr ON po.srcbillnumber = pr.billno
JOIN public_gitea_analytics.erp_mrp_plan_order mrp ON pr.srcbillnumber = mrp.billno
WHERE mrp.rootdemandbillno = 'YCD2026022800000048';
```

### A.5 工单查询

```sql
SELECT billno, material_number, material_name, qty, xkquainwaqty, taskstatus_title
FROM public_gitea_analytics.erp_production_work_order
WHERE material_number IN ('T01-000080', '723-000002', '948-000077')
AND billdate >= '2026-03-01';
```

### A.6 库存查询

```sql
SELECT material_code, material_name, warehouse, inventory_qty, available_inventory_qty
FROM public_gitea_analytics.erp_real_time_inventory
```
